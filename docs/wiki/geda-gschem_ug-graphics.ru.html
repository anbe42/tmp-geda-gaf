Графические примитивы

В gschem используется ряд чисто графических элементов. Они, например, могут использоваться для черчения символов, блок-схем или заметок на полях схем, для графического выделения отдельных функциональных частей большого проекта. Программой gnetlist они игнорируются.

Если графический элемент выделен, для него отображается одна или несколько ручек управления, соответствующих положению контрольных точек объекта. Их можно использовать для изменения элемента, для этого нужно щёлкнуть по ручке левой кнопкой мыши и, удерживая кнопку нажатой, переместить её в другое место.

Если графический элемент создаётся с помощью привязки клавиш, место расположения первой контрольной точки определяется текущей позицией курсора мыши.

Линии

Простые прямые линии можно добавить на страницу с помощью Добавить→Линию [Add→Line].

Прямоугольники

Прямоугольники добавляются на страницу с помощью Добавить→Прямоугольник [Add→Box]. Для создания прямоугольника надо указать две его вершины, расположенные друг напротив друга по диагонали.

Дуги

Дуги добавляются с помощью Добавить→Дугу [Add→Arc]. Щелчком левой кнопки мыши выбирается точка центра дуги, при этом откроется окно “Параметры дуги” [“Arc Params”]. В нём можно задать радиус, начальный угол и угол разворота дуги. Для создания дуги нужно щёлкнуть OK.

Для редактирования дуг используется три контрольных точки. Для изменения радиуса дуги используется ручка, расположенная в центре. Две другие, внешние ручки предназначены для корректировки начального угла дуги и угла её разворота. Чтобы открыть окно “Параметры дуги” ещё раз для повторного редактирования, дугу нужно выделить и в меню выбрать Правка→Правка… [Edit→Edit…].

Окружности

Окружности на страницу можно добавить с помощью Добавить→Окружность [Add→Circle]. Первым щелчком левой кнопки мыши выбирается положение центра окружности, а вторым задаётся её радиус. Для изменения окружности используется только одна контрольная точка, предназначенная для изменения её радиуса.

Текстовые объекты

Добавить текст на страницу можно с помощью Add→Text [Add→Text]. При этом открывается диалоговое окно “Ввод текста” [“Text Entry…”], где можно ввести несколько строк текста. Для вставки текста надо щёлкнуть кнопку Применить [Apply], а затем щёлкнуть левой кнопкой в области просмотра схемы для добавления текстового объекта на страницу.

С помощью управляющей последовательности ”\_”, которая сама по себе на странице отображаться не будет, можно добавить горизонтальную черту над текстом. Например, буква “B” в тексте ”CS/\_B\_” будет отображаться с горизонтальной чертой над нею. Если текстовый объект имеет вид ”<имя>=<значение>”, то он является атрибутом.

Чтобы изменить текст, его нужно выделить и в меню выбрать Правка→Правка текста… [Edit→Edit Text…], в результате чего откроется окно Правка свойств текста [Edit Text Properties]. В этом окне можно отредактировать как сам текст, так и его цветовой индекс, размер шрифта и выравнивание.

Настройки выравнивания изменяют положение точки привязки текстового объекта относительно самого текста. Чтобы увидеть, где находится точка привязки текстового объекта, его нужно выделить и увеличить масштаб отображения. Тогда можно увидеть маленький маркер “X”, отмечающий точку привязки.

Изображения

Объект изображение служит для отображения на страницах схем и символов изображений из внешних файлов. Изображение добавляется с помощью Добавить→Изображение [Add→Picture]. При этом открывается окно выбора файла, где нужно выбрать добавляемый файл. Если выбранный файл не может быть загружен в gschem, появится сообщение об ошибке.

Когда файл изображения выбран, сначала нужно щёлкнуть левой кнопкой в месте его размещения, а затем щёлкнуть ещё раз, чтобы выбрать его размер.

Для изменения размера изображения, его надо выделить и потянуть одну из его ручек. Отношение сторон изображения при этом автоматически будет поддерживаться правильным. Для замены изображения нужно выбрать Правка→Правка… [Edit→Edit], что приведёт к открыванию окна выбора файла, в котором можно выбрать нужный файл изображения.

Как и компоненты, изображения как объекты могут быть либо ссылками на файлы изображений, либо файл изображения может быть целиком скопирован (“внедрён”) в файл схемы или символа. По умолчанию объект изображение ссылается на файл изображения. Изображение можно внедрить с помощью Правка→Внедрить компонент/изображение [Edit→Embed Component/Picture], или произвести обратное действие с помощью Правка→Исключить компонент/изображение [Edit→Unembed Component/Picture].

Если внедрённое изображение не соответствует в полной мере имеющемуся файлу изображения, выбор Правка→Исключить компонент/изображение не приведёт к его изменению.
