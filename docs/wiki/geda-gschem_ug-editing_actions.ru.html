Действия правки

Для редактирования схем и символов в основном окне gschem используются действия правки. Вызвать действие правки можно одним из четырёх способов: с клавиатуры, используя одно или несколько нажатий клавиш; из основного меню; с панели инструментов; из контекстного меню.

Многие действия работают с текущим выделением. См. раздел Выделение объектов.

Если результат действия правки вам не нравится, его можно отменить нажатием <Ctrl+Z> или с помощью Правка→Отменить [Edit→Undo]. Чтобы вернуть результаты отменённого действия, нажмите <Ctrl-Y> или выберите в меню Правка→Повторить [Edit→Redo].

Привязки клавиш

Каждому действию соответствует определённая привязка клавиш (keybinding). Привязка определяется последовательностью нажатий сочетаний клавиш. Например, вызов документации gEDA осуществляется с помощью привязки <H M>: нужно нажать сначала клавишу <H>, а затем <M>.

Иногда в привязки входят клавиши-модификаторы. Например, чтобы начать добавление “шин” в схему, используется привязка <Shift+B>: нужно нажать клавишу <B>, удерживая нажатой одну из клавиш <Shift> на клавиатуре.

Если нажать сочетание клавиш, оно кратковременно отразится в строке состояния и пропадёт, если только оно не является началом многоклавишной привязки, в этом случае названия клавиш будут отображаться в строке состояния до нажатия другой клавиши. Например, если нажать <H>, в строке состояния появится буква “H”. Если затем нажать <M>, чтобы открыть руководство, в строке состояния кратковременно отобразится “H M”, а затем эти буквы пропадут.

В окне “Горячие клавиши” [“Hotkeys”] (доступном из меню Справка→Горячие клавиши [Help→Hotkeys…]) отображаются все текущие привязки клавиш и связанные с ними действия. Привязки клавиш легко изменить в файлах настроек. Более подробную информацию об этом см. в разделе Настройка gschem.

Основное меню и панель инструментов

Чтобы вызвать действие из основного меню или с панели инструментов, нужно просто щёлкнуть по соответствующему элементу левой кнопкой мыши. Обратите внимание, если навести курсор мыши на одну из кнопок панели инструментов, появится подсказка по соответствующему инструменту, содержащая более подробную информацию о действии.

В каждом пункте основного меню показана эквивалентная ему привязка клавиш. Это позволяет изучать клавиши для вызова определённых действий, не прибегая каждый раз к окну “Горячие клавиши”.

Контекстные меню

Если щёлкнуть в области отображения схемы основного окна правой кнопкой мыши, появится контекстное меню. Данное меню содержит несколько общеупотребительных действий правки.
