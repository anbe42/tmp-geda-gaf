Printing

gschem can output PostScript files suitable for printing by using File→Print…. This displays the “Print…” window.

Here, you can choose the paper size and orientation to use, and what area of the schematic or symbol to print. There are three options for the “Type” of print:

  • If “Extents with margins” is selected, everything visible will be printed, with a margin automatically added on all sides of the page.
  • “Extents no margins” also prints all visible objects, except without allowing any margin.
  • With “Current view”, only the objects currently visible in the schematic view area of the main window will be printed.

Output can be directed either to a PostScript file, or to a command (such as lpr on Unix systems).
