Аргументы командной строки

gschem принимает несколько дополнительных аргументов командной строки. Их перечень можно узнать, запустив gschem -h. Перечень всех ключей командной строки, принимаемых gschem, можно посмотреть в man-странице программы (man gschem).

Вывод сообщений и диагностика

-q, --quiet Без вывода информации. Отключение всех предупреждений/замечаний/сообщений.
-v, --verbose Подробный вывод. Вывод дополнительной диагностической информации.
-h, --help Вывод справки.
-V, --version Вывод информации о версии.

Использование расширений на Scheme

-L КАТАЛОГ Добавление КАТАЛОГа к началу списка каталогов, в которых производится поиск файлов Scheme.
-s ФАЙЛ Задание скрипта на Scheme, который должен быть выполнен при запуске.
-c ВЫРАЖЕНИЕ Задание выражения на Scheme, которое должно быть выполнено при запуске.
-- Считать все остальные аргументы именами файлов схем и символов. Это может использоваться, когда нужно задать имя файла схемы или символа, начинающееся с «-».

Прочее

-r, --config-file=ФАЙЛ Задание дополнительного файла настройки, который должен быть считан после обработки всех прочих файлов настроек.
-o, --output=ФАЙЛ Задание имени файла для вывода в формате PostScript. Этот аргумент полезен при запуске gschem из скрипта оболочки с заданием выполнения скрипта на Scheme. Имя файла может быть изменено в диалоговом окне печати.
-p Автоматическое размещение окна. Может быть полезно, если нужно запустить gschem из командной строки и вывести результаты в файл.
