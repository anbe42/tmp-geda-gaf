Перемещение и копирование

Перемещение объектов

Для перемещения объекта существует три способа:

  1. Щёлкнуть по нему левой кнопкой мыши и потащить. Если он выделен, с ним будут перемещаться и все остальные выделенные объекты.
  2. Выделить один или несколько объектов и нажать <M> для начала перемещения, а затем щёлкнуть левой кнопкой мыши для размещения их в новой позиции.
  3. Выделить один или несколько объектов и использовать Правка→Перемещение [Edit→Move Mode]. Щёлкнуть один раз, чтобы начать перемещение, и ещё раз, чтобы его завершить.

Все соединения, подключенные к перемещаемому объекту, будут растягиваться (как “резиновые”) для сохранения связей между объектами. Такое поведение можно включать и отключать с помощью Настройки→Резиновое соединение вкл/откл [Options→Toggle net rubberband].

Если при перемещении удерживать нажатой клавишу <Ctrl>, направление перемещения будет ограничено — смещать объекты можно будет только по горизонтали или по вертикали относительно их исходного положения.

Поворот и зеркальное отражение объектов

Объекты или их группы можно также поворачивать и отражать относительно вертикальной оси.

Для поворота выделенных объектов на 90° против часовой стрелки относительно текущего положения курсора мыши нажмите <E R>. В процессе размещения объекта на странице (например, при его перемещении), повернуть его можно также щелчком средней кнопки мыши.

Чтобы отразить выделенные объекты слева направо относительно положения курсора мыши, нажмите <E I>.

Копирование объектов в пределах страницы

Быстро скопировать выделенные объекты можно, нажав <E C>, а затем щёлкнув левой кнопкой для размещения копии. Щелчком правой кнопки мыши начатое копирование можно отменить. Быстро скопировать объекты можно и выбрав в меню Правка→Копирование [Edit→Copy mode], в этом случае для начала копирования нужно будет щёлкнуть левой кнопкой мыши, а затем щёлкнуть той же кнопкой ещё раз для его завершения.

Если нужно сделать более одной копии, можно использовать Правка→Многократное копирование [Edit→Multiple Copy Mode], или нажать <E Y>. Это почти ничем не отличается от обычного быстрого копирования, разница только в том, что, щёлкая несколько раз левой кнопкой мыши, можно сделать несколько копий выделенных объектов, а завершается копирование щелчком правой кнопки.

Если при быстром копировании удерживать нажатой клавишу <Ctrl>, копии будут выравниваться по исходным объектам по горизонтали или по вертикали.

Копирование и вставка с помощью буфера обмена

Выделенные объекты можно скопировать в системный буфер обмена, а позже их оттуда вставлять. Чтобы скопировать выделенные объекты в буфер, в меню выберите Правка→Копировать [Edit→Copy] или, чтобы поместить их в буфер обмена, удалив со страницы, — Правка→Вырезать [Edit→Cut].

Если в системном буфере находятся данные схемы или символа gEDA, их можно добавить на страницу с помощью Правка→Вставить [Edit→Paste]. Отменить операцию в процессе помещения данных из буфера можно щелчком правой кнопки.

Использование буферов

В gschem поддерживается несколько буферов, работа с которыми ведётся почти также, как и с системным буфером обмена, но с одним исключением — для других приложений данные из них недоступны. Действия, относящиеся к работе с этими буферами, можно найти в меню Буфер [Buffer].

Использовать буферы может быть удобно в тех случаях, когда хочется иметь под рукой не один объект или группу объектов, а несколько, и вставлять их не все сразу, а по отдельности.
